# AdventureWorks Executive BI Dashboard

## Project Overview
This project analyzes **$24.9M in revenue** and **$10.5M in profit** using a retail dataset. I transformed raw data into an interactive executive dashboard to track global performance.

## Key Technical Features
* **Data Modeling:** Built a **Star Schema** with one-to-many relationships across 7+ tables.
* **DAX Intelligence:** Created measures for **Year-over-Year Growth**, **10-day Rolling Revenue**, and **What-If Pricing Simulations**.
* **Visual Storytelling:** Developed pages for Executive Overview, Map Analysis, and Customer Details.

## Data Architecture

*The model uses a central Sales fact table connected to Product, Customer, and Calendar dimension lookups.*
